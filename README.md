# ML_labs
**lab_1:** На основе данных о квартирах в городе N подготовить данные, исследовать влияние факторов на стоимость, выделить группы квартир, а затем построить модель для предсказания цены (или цены за квадратный метр) и ценовых категорий.
**lab_2:** Разработать и оценить модели машинного обучения для предсказания оттока клиентов банка с учётом дисбаланса классов, чтобы своевременно выявлять клиентов с высоким риском ухода и предоставлять бизнесу инструмент для повышения удержания клиентов и оптимизации маркетинговых затрат. В качестве базовой модели использовалась LogisticRegression для оценки исходного качества и последующего сравнения с моделями, учитывающими дисбаланс классов (RandomForest, LightGBM, CatBoost).
**lab_3:** Используя предоставленные социально-экономические и демографические данные по странам, провести сравнительный кластерный анализ для объективного выявления групп стран с наиболее критическим уровнем развития и нуждаемости. На основе результатов анализа сформировать приоритетный список стран для целевой финансовой и гуманитарной поддержки международной благотворительной организации.
**lab_4:** Провести комплексный анализ кредитного риска клиентов банка путем построения, оптимизации и сравнительной оценки различных моделей машинного обучения — от простого дерева решений до сложных ансамблевых алгоритмов. Основная практическая задача — создать высокоточный классификатор для прогнозирования серьезных просрочек платежей (Delinquent90), а также выявить ключевые факторы кредитного риска и оценить вклад признаков.
**lab_5:** Провести комплексный анализ и построить прогнозную модель для месячного временного ряда, отражающего численность сотрудников в индустрии гостеприимства штата Калифорния. Работа направлена на выявление ключевых закономерностей данных (тренд, сезонность) и создание точного статистического прогноза на будущие периоды для поддержки стратегического планирования в сфере туризма и гостеприимства.
**lab_6:** Разработать и оптимизировать модель глубокого обучения для прогнозирования рейтинга фильма на КиноПоиске на основе его атрибутивной информации (бюджет, государственная поддержка, страна, студия и т.д.) и финансовых показателей сборов. Проект направлен на изучение влияния различных технических и бизнес-факторов на успех кинокартины у зрителей, а также на освоение полного цикла построения и тонкой настройки нейронной сети.
**lab_7:** Разработать и реализовать с нуля (без использования высокоуровневых абстракций современных фреймворков глубокого обучения) архитектуру, алгоритм прямого и обратного распространения (forward/backward pass), а также процесс обучения полносвязной нейронной сети для решения задачи классификации или регрессии. Проект направлен на глубокое понимание фундаментальных принципов работы нейронных сетей: линейных преобразований, функций активации, градиентного спуска и механизма обратного распространения ошибки.
**lab_8:** Освоить и продемонстрировать на практике метод трансферного обучения (Transfer Learning) — ключевую технику в компьютерном зрении. Проект направлен на эффективное решение задачи классификации спутниковых снимков земной поверхности (датасет EuroSAT) путем адаптации мощной предобученной свёрточной нейронной сети, изначально обученной на крупном наборе общих изображений, к новой, узкоспециализированной предметной области.
**lab_9:** Реализовать, обучить и продемонстрировать работоспособность кастомной архитектуры детектора объектов на основе фреймворка YOLO для задачи автоматического обнаружения опухолей головного мозга на  медицинских снимках из датасета Brain-tumor. Проект нацелен на практическое освоение принципов построения современных детекторов: использования предобученных классификационных сетей в качестве экстракторов признаков, реализации комбинированной функции потерь и применения постобработки с помощью алгоритма Non-Maximum Suppression.
